# ExpertNotes

Este é um projeto para registrar notas, permitindo a entrada de texto escrito ou a conversão de áudio para texto. O projeto utiliza uma variedade de tecnologias modernas para garantir uma experiência de usuário rica e responsiva.

O projeto está rodando aqui para testar: https://expertynote.netlify.app/

## Tecnologias Utilizadas

- **ReactJS**: Biblioteca JavaScript para a construção de interfaces de usuário.
- **TypeScript**: Superset de JavaScript que adiciona tipagem estática ao código.
- **TailwindCSS**: Framework de CSS para estilização rápida e eficiente.
- **Vite**: Ferramenta de construção que proporciona um ambiente de desenvolvimento rápido.
- **HTML5**: A última versão do HTML para a estruturação das páginas web.
- **CSS3**: A última versão do CSS para estilização das páginas web.

## Pacotes e Extensões Utilizados

- **Radix UI**: Conjunto de componentes acessíveis e descomplicados para construção de interfaces de usuário.
- **date-fns**: Biblioteca para manipulação de datas em JavaScript e TypeScript.
- **lucide-react**: Ícones SVG para React.
- **SpeechRecognition**: API para reconhecimento de voz e conversão de áudio para texto.

## Funcionalidades

- **Registro de Notas Escritas**: Permite ao usuário digitar e salvar notas.
- **Conversão de Áudio para Texto**: Permite ao usuário gravar áudio que será convertido em texto e salvo como nota.
- **Interface Intuitiva**: Interface de usuário moderna e responsiva, construída com Radix UI e TailwindCSS.
- **Manipulação de Datas**: Utilização do date-fns para formatação e manipulação de datas.
- **Ícones**: Ícones SVG fornecidos pelo lucide-react para uma melhor representação visual.
